v1.0.0 - 初回リリース。時間帯スケジュールによるブルーライトカット、タスクトレイ常駐、設定画面を実装。
v1.0.1 - 日本語・英語の多言語対応を追加。ナビゲーションの情報・設定をサイドバー下部へ移動。UIスタイル名の整理。
v1.0.2 - アプリ名を BlueShift に変更。アイコンを設定。実行ファイル名を BlueShift.exe に変更。
v1.0.3 - タスクトレイからウィンドウが再表示できない不具合を修正。WinUI ウィンドウの WndProc 差し替えを廃止。
v1.0.4 - タスクトレイ用ウィンドウ作成失敗による起動クラッシュを修正。
v1.0.5 - 起動クラッシュ回避のためタスクトレイを安定方式に戻し、ウィンドウ再表示の改善は維持。
v1.0.6 - タスクトレイ非表示時にタスクバーへ残る問題を修正。二重起動で既存ウィンドウを表示。GitHub リリース URL を設定。スケジュール未設定時のガンマ適用を抑制。
v1.0.7 - 強制終了後に残るガンマ補正のリセット処理を追加（--reset-gamma）。起動直後のウィンドウ非表示を修正。終了時のガンマ復元を強化。
v1.0.8 - v1.0.7 の Activated 待ちによる起動時の画面非表示を修正。起動直後からタスクトレイアイコンを常時表示。
v1.0.9 - --reset-gamma を WinUI 起動前に実行するよう変更。プロセスなしで残るガンマ補正を即座に解除可能に。
v1.0.10 - WinUI ウィンドウの WndProc 差し替えを廃止し白画面を修正。タスクトレイ専用ウィンドウを再導入。ResetBlueShiftGamma.bat のパスエラーを改善。
v1.0.11 - Win32 ShowWindow 呼び出しを廃止し AppWindow API のみで表示。MicaBackdrop を削除。UI 描画完了後にガンマ適用。
v1.0.12 - 起動時白画面を修正。OnLaunched で Activate を追加。UI 準備完了後にページ表示。初回フレーム描画後にガンマ適用。HWND 操作を起動時から除去。
v1.0.13 - 起動フローを根本再設計。Loaded/Activated 二重待ちを廃止。UI 初回描画後にのみタスクトレイ・ガンマ・バックグラウンド非表示を実行。二重起動時は GUI 表示要求を明示的に通知。
v1.0.14 - Loaded 中の UpdateLayout 呼び出しを削除し MeasureOverride 例外を修正。ページ遷移を次フレームへ defer。PageScrollHost の Width 直接変更を MaxWidth に変更。
v1.0.15 - PageScrollHost を静的レイアウトに簡素化。TimePage の二重 ScrollViewer を解消。生成コードの Debugger.Break を無効化し App 側で例外を記録。
v1.0.16 - Time_Delete の x:Uid ツールチップキーを ToolTipService.ToolTip に修正し XamlParseException を解消。
v1.0.17 - v1.0.17〜v1.0.18 の UI 変更（カード統一・トグルラベル・スクロール制御・ホイールスライダー・ガンマ遷移・プレビュー）を取り消し v1.0.16 相当へ戻した。
v1.0.18 - 設定タブのみ UI を時間タブと統一（ラベル左配置）。スクロール自動無効化・ホイールスライダー・なめらかなガンマ遷移・ドラッグ時プレビューを追加。時間・情報タブの UI は変更なし。
v1.0.19 - 全タブを Auto Dark Mode と同方式に統一。CommunityToolkit SettingsCard 採用、PageBaseStackPanelStyle（MaxWidth 1000・余白 24,0,24,24）を全ページ共通適用。
v1.0.20 - スライダーのマウスホイール操作時はガンマプレビューを行わず、強度の数値調整のみに変更。
v1.0.21 - 複数スケジュール時の強度誤適用を修正。時刻判定ロジックを改善し、ListView スライダーを Id 紐付け・TwoWay バインドに変更。
v1.0.22 - タブ切替後の強度ずれを修正（プレビューはドラッグ時のみ）。ログオン自動起動時は GUI を表示せずタスクトレイ常駐のみに変更。
